# mdFileWriter - 使い方ガイド

## 概要
**mdFileWriter** は、Markdown記法を意識せず **Word感覚**でMarkdownファイル（`.md`）を作成・編集・保存できるデスクトップアプリです。  
Electronベースで動作し、Windows 10/11 でインストーラまたはポータブル実行ファイルとして起動できます。

---

## 起動方法

### インストーラ版
1. `MdFileWriter Setup 1.3.0.exe` を実行してインストールする。
2. スタートメニューまたはデスクトップショートカットから起動する。

### ポータブル版
1. `MdFileWriter 1.3.0.exe` を任意のフォルダに配置する。
2. ファイルをダブルクリックして起動する。

## 基本操作

### ファイルを開く（ドラッグ＆ドロップ）
1. 編集したい `.md` ファイルをエクスプローラーから左ペインの **D&Dゾーン**（破線ボーダーのエリア）にドラッグ＆ドロップする。
2. 左ペインにファイル名が表示され、右ペインのエディタに内容が読み込まれる。
3. テキストを直接編集する（Markdownを意識する必要はない）。

> **注意**: 開けるのは `.md` ファイルのみです。他の形式のファイルはエラーになります。

### ファイルを保存する
- ツールバーの **「💾 保存」** ボタンをクリックする。
- または **Ctrl+S** を押す。
- 未保存の場合はタイトルバーに `*` が表示される。

### 新規ファイルを作成する
1. 現在ファイルをD&Dで開いた後、ツールバーの **「📄」** ボタンをクリックする。
2. 現在開いているファイルと同じフォルダにタイムスタンプ付きの新規 `.md` ファイルが作成される。

### ファイルを削除・リネームする
- ツールバーの **「🗑️」** ボタン: 確認ダイアログ後に現在開いているファイルが削除される。
- ツールバーの **「✏️」** ボタン: 新しいファイル名を入力して名前変更する。

## ツールバー操作

| ボタン | 内容 |
|:--|:--|
| 💾 保存 | 現在のファイルをUTF-8で上書き保存 |
| 📄 新規作成 | 同フォルダに新規 `.md` ファイルを作成 |
| 🗑️ 削除 | 現在開いているファイルを削除 |
| ✏️ リネーム | 現在開いているファイルの名前を変更 |
| H1 / H2 | 見出し（1段・2段）を適用 |
| **B** (太字) | 選択テキストを太字にする |
| *I* (斜体) | 選択テキストを斜体にする |
| ~~S~~ (取り消し線) | 選択テキストに取り消し線を付ける |
| 箇条書きリスト | 箇条書きを挿入する |
| 番号付きリスト | 番号付きリストを挿入する |
| ☑ タスクリスト | チェックボックス付きリストを挿入する（`- [ ]` / `- [x]`） |
| 🔗 リンク | リンクを挿入する |
| 🖼️ 画像 | 画像を挿入する |
| 📅 表 | テーブルを挿入する |
| 「{}」コード | コードブロックを挿入する |
| " 引用 | 引用ブロックを挿入する |
| ― 水平線 | 水平線（区切り線）を挿入する |

## 編集モードの切替
右ペイン右上のドロップダウンメニューで編集モードを切り替えることができます。

- **WYSIWYGモード**（デフォルト）: Wordのように見た目通りに編集できる。
- **Markdownソースモード**: Markdownのテキストを直接編集できる。

---

## 未保存の確認
- 編集中に別のMDファイルをD&Dしようとしたとき

未保存の変更がある場合は確認ダイアログが表示されます。「OK」で破棄して開く、「キャンセル」で現在の編集を維持できます。

---

## 注意事項
- 保存形式は常に **UTF-8** です。
- 行末スペース2個（`  `）は CommonMark 仕様の強制改行（`<br>`相当）として扱われます。ファイル読み込み時に自動正規化されるため、3スペース以上の行末スペースも２スペースに統一されます。WYSIWYGモードでも `<br>` として視覚表示されます。
- 画像は **ローカルファイルへのパス参照** となります（ファイルへの埋め込みは非対応）。
- パストラバーサル（`../`）を含むパスはセキュリティ上拒否されます。
