# mdFileWriter - 使い方ガイド

## 概要
**mdFileWriter** は、Markdown記法を意識せず **Word感覚**でMarkdownファイル（`.md`）を作成・編集・保存できるデスクトップアプリです。  
Electronベースで動作し、Windows 10/11 でインストーラまたはポータブル実行ファイルとして起動できます。

---

## 起動方法

### インストーラ版
1. `MdFileWriter Setup 1.0.1.exe` を実行してインストールする。
2. スタートメニューまたはデスクトップショートカットから起動する。

### ポータブル版
1. `MdFileWriter 1.0.1.exe` を任意のフォルダに配置する。
2. ファイルをダブルクリックして起動する。

## 基本操作

### フォルダを開く
1. 左ペインの **「フォルダを開く」** ボタンをクリックする。
2. 表示したいMarkdownファイルが入ったフォルダを選択する。
3. 選択フォルダ配下の `.md` ファイルがツリーに表示される。

### ファイルを開く・編集する
1. 左ペインのファイルツリーからファイルをクリックする。
2. 右ペインのエディタにファイル内容が表示される。
3. テキストを直接編集する（Markdownを意識する必要はない）。

### ファイルを保存する
- ツールバーの **「保存」** ボタンをクリックする。
- または **Ctrl+S** を押す。
- 未保存の場合はタイトルバーに `*` が表示される。

### 新規ファイルを作成する
1. ファイルツリー上部の **「新規作成」** ボタンをクリックする。
2. ファイル名を入力してOKを押す。
3. 現在開いているフォルダ内に新しい `.md` ファイルが作成される。

### ファイルを削除・リネームする
- ファイルツリーでファイルを右クリック（またはコンテキストメニューを開く）。
- **「削除」**: 確認ダイアログ後にファイルが削除される。
- **「リネーム」**: 新しいファイル名を入力してファイル名を変更する。

## ツールバー操作

| ボタン | 内容 |
|:--|:--|
| 見出し (H1〜H6) | ドロップダウンで見出しレベルを選択 |
| **B** (太字) | 選択テキストを太字にする |
| *I* (斜体) | 選択テキストを斜体にする |
| ~~S~~ (取り消し線) | 選択テキストに取り消し線を付ける |
| 箇条書きリスト | 箇条書きを挿入する |
| 番号付きリスト | 番号付きリストを挿入する |
| リンク | リンクを挿入する |
| 画像 | 画像を挿入する |
| 表 | テーブルを挿入する |
| コードブロック | コードブロックを挿入する |
| 引用 | 引用ブロックを挿入する |
| 水平線 | 水平線（区切り線）を挿入する |
| Undo | 直前の操作を元に戻す |
| Redo | 元に戻した操作をやり直す |

## キーボードショートカット

| ショートカット | 操作 |
|:--|:--|
| **Ctrl+S** | 保存 |
| **Ctrl+B** | 太字 |
| **Ctrl+I** | 斜体 |
| **Ctrl+Z** | 元に戻す (Undo) |
| **Ctrl+Y** | やり直し (Redo) |

---

## 編集モードの切替
右ペイン上部のタブで編集モードを切り替えることができます。

- **WYSIWYGモード**（デフォルト）: Wordのように見た目通りに編集できる。
- **Markdownソースモード**: Markdownのテキストを直接編集できる。

---

## 未保存の確認
- 編集中にファイルを切り替えようとしたとき、または
- アプリを閉じようとしたとき

未保存の変更がある場合は確認ダイアログが表示されます。「保存」「破棄」「キャンセル」から選択してください。

---

## 注意事項
- 操作できるファイルは **選択したフォルダ配下のみ** に限定されます（セキュリティ仕様）。
- 保存形式は常に **UTF-8** です。
- 画像は **ローカルファイルへのパス参照** となります（ファイルへの埋め込みは非対応）。
